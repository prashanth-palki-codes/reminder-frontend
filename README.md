## Tech Stack & Packages

- **React + TypeScript**
  - UI development

- **Vite**
  - Fast development and build tooling

- **Redux Toolkit**
  - Authentication and reminders state management

- **React Router**
  - Client-side routing and protected routes

- **Tailwind CSS**
  - Styling and layout

- **Axios**
  - API communication with backend

- **SweetAlert2**
  - Confirmation dialogs (logout, delete)

- **React Hot Toast**
  - User feedback notifications

---

## Key Features

- Login and registration
- Create, edit, delete reminders
- Upcoming and past reminders view
- Frontend date-based filters (week / month)
- Account details update
- Confirmation prompts for critical actions
- Clean and responsive UI
